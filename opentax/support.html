OpenTax 지원

OpenTax는 공개 온톨로지와 읽기 전용 MCP endpoint를 제공합니다. 오류, 누락된 기준금액, 잘못된 출처, ChatGPT 연결 문제는 GitHub Issues로 접수합니다.

지원 채널

  • GitHub Issues
  • TaxMeter repository

서비스 URL

  • OpenTax web: https://jhny-kor.github.io/TaxMeter/opentax/
  • MCP endpoint: https://opentax-mcp.y2kthr.workers.dev/mcp
  • Health check: https://opentax-mcp.y2kthr.workers.dev/health

ChatGPT 테스트 문장

  • OpenTax에서 보험료 공제 한도를 찾아서 출처와 함께 설명해주세요.
  • OpenTax에서 ISA 비과세 한도와 근거를 찾아주세요.
  • OpenTax에서 근로장려금 신청 기준을 조회하고 공식 출처를 보여주세요.

알려진 제한

OpenTax는 개인별 세액을 계산하거나 신고서를 작성하지 않습니다. 결과는 공식 출처를 찾기 위한 참고 정보이며, 실제 신고와 신청 전에는 관련 기관 또는 전문가 확인이 필요합니다.
