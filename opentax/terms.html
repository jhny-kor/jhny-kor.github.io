OpenTax 이용약관

OpenTax는 대한민국 세금, 공제, 정책지원, 신고기한, 공식 출처 정보를 탐색하기 위한 공개 지식 그래프와 읽기 전용 MCP 서비스를 제공합니다.

정보 제공의 성격

OpenTax는 일반 정보와 공식 출처 연결을 제공하며, 세무, 법률, 금융, 회계 자문을 제공하지 않습니다. 실제 신고, 납부, 신청, 불복, 투자, 대출 판단은 국세청, 지방자치단체, 금융기관, 세무사, 변호사 등 적절한 전문가 또는 공식 기관을 통해 확인해야 합니다.

데이터 기준

OpenTax의 기준일, 출처, 금액, 세율, 한도는 온톨로지 노드에 표시된 basis date와 source를 따릅니다. 세법과 정책지원 제도는 수시로 변경될 수 있으므로 결과에 표시된 공식 출처를 함께 확인해야 합니다.

허용되는 사용

  • 세목, 공제, 지원금, 신고기한, 공식 출처의 검색과 조회
  • 공식 출처를 바탕으로 한 학습, 비교, 문서화, 앱 개발 참고
  • OpenTax JSON export와 MCP endpoint의 읽기 전용 사용

제한

  • OpenTax를 정부기관, 국세청, OpenAI가 보증하거나 운영하는 서비스처럼 표시하지 마세요.
  • OpenTax 결과만으로 개인 세무 신고나 금융 거래를 확정하지 마세요.
  • 서비스 남용, 과도한 자동 요청, 보안 우회, 출처 왜곡을 하지 마세요.

문의

오류 신고와 개선 제안은 GitHub Issues로 접수합니다.
